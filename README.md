# GreenSpark ePortfolio GitHub Pages Site

This package contains a static website for José Silva's sustainability ePortfolio.

## Files
- `Eblog portfolio.html` = homepage
- `portfolio1.html` through `portfolio7.html` = portfolio pages
- `downloads/` = original DOCX files preserved exactly as submitted
- `assets/` = supporting images for Portfolio 7

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Save.
6. GitHub will publish your site and give you a public URL.

## Notes
- The website is more visual than embedded documents.
- The original portfolio DOCX files are included so no information is lost.
